# Leather.io Web App

Home of Leather's web app, built with [React Router v7](https://reactrouter.com/) in framework mode.

## Getting Started

### Installation

```bash
pnpm install
```

### Development

Start the development server with HMR:

```bash
pnpm dev
```

Your application will be available at `http://localhost:5173`.
